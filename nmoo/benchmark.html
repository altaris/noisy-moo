nmoo.benchmark

A benchmarking utility. The following is perhaps the one of simplest benchmark one could make

from pymoo.problems.multi import ZDT1
from pymoo.algorithms.moo.nsga2 import NSGA2
from nmoo import Benchmark, GaussianNoise, KNNAvg, WrappedProblem

zdt1 = WrappedProblem(ZDT1())
noisy_zdt1 = GaussianNoise(zdt1, np.zeros(2), 1)
knnavg_zdt1 = KNNAvg(noisy_zdt1, max_distance=1.0)

benchmark = Benchmark(
    problems={
        "knnavg": {
            "problem": knnavg_zdt1
        }
    },
    algorithms={
        "nsga2": {
            "algorithm": NSGA2()
        }
    },
    n_runs=3,
    output_dir_path="./out",
)

which simply runs vanilla NSGA2 against a KNN-Averaging-denoised Gaussian-noised synthetic ZDT1 against NSGA21, 3 times. The benchmark can be executed with

benchmark.run()

and the ./out directory will be populated with various artefact, see below.

Refer to https://github.com/altaris/noisy-moo/blob/main/example.ipynb to get started, or to https://github.com/altaris/noisy-moo/blob/main/example.py for a more complete example.

  • <problem>.<algorithm>.<run number>.<layer number>-<layer name>.npz: NPZ archive containing the history of all calls to a given layer of a given problem. In the example above, problem knnavg_zda1 has three layers: knn_avg (layer 1, the outermost one), gaussian_noise (layer 2), and wrapped_problem (layer 3, the innermost one). Recall that you can set the name of a layer using the name argument in WrappedProblem.__init__. The keys are X, F, _batch, _run. It may also contain keys G, dF, dG, ddF, ddG, CV, feasible depending on the ground pymoo problem. The arrays at each keys have the same length (shape[0]), which is the number of individuals that have been evaluated throughout that run. In our example above, out/knnavg.nsga2.1.1-knn_avg.npz has keys X, F, _batch, _run, and the arrays have shape (19600, 30), (19600, 2), (19600,), (19600,), respectively. 30 is the number of variables of ZDT1, while 2 is the number of objectives.

  • <problem>.<algorithm>.<run number>.pp.npz: Pareto population of a given (algorithm, problem, run number) triple. The keys are X, F, G, dF, dG, ddF, ddG, CV, feasible, _batch, and all arrays have the same length (shape[0]). Row i corresponds to an individual that was Pareto-ideal at generation _batch[i].

  • <problem>.<algorithm>.gpp.npz: Global Pareto population of a given problem-algorithm pair. It is the Pareto population of the population of all individuals designed across all runs and all generations of a given problem-algorithm pair. It is used to compute certain performance indicators in the absence of a baseline Pareto front. The keys are X, F, G, dF, dG, ddF, ddG, CV, feasible, _batch.

 171class PARTriple(PAPair):
 172    """
 173    Represents a problem-algorithm-(run number) triple.

 178    def __str__(self) -> str:

 181    def denoised_top_layer_history_filename(self) -> str:
 182        """Returns
 183        `<problem_name>.<algorithm_name>.<n_run>`.1-<top_layer_name>.denoised.npz`.
 184        """
 185        prefix = self.filename_prefix()
 186        name = self.problem_description["problem"]._name

 188
 189    def filename_prefix(self) -> str:
 190        """Returns `<problem_name>.<algorithm_name>.<n_run>`."""
 191        return f"{self.problem_name}.{self.algorithm_name}.{self.n_run}"
 192
 193    def innermost_layer_history_filename(self) -> str:
 194        """Returns the filename of the innermost layer history."""
 195        prefix = self.filename_prefix()
 196        problem = self.problem_description["problem"]
 197        inner = problem.innermost_wrapper()
 198        name, depth = inner._name, problem.depth()
 199        return f"{prefix}.{depth}-{name}.npz"

 202        """Returns `<problem_name>.<algorithm_name>.<n_run>.pp.npz`."""
 203        return self.filename_prefix() + ".pp.npz"

 205    def pi_filename(self, pi_name: str) -> str:
 206        """
 207        Returns `<problem_name>.<algorithm_name>.<n_run>.pi-<pi_name>.csv`.
 208        """

 420                * `gigd+`: ground inverted generational distance plus; requires
 421                  `pareto_front` to be set in the problem description
 422                  dictionaries, otherwise the value of this indicator will be
 423                  `NaN`.
 424                * `rggd`: resampled ground generational distance, where the
 425                  ground problem's predicted objective values (resampled and
 426                  averaged a given number of times) are used instead of the
 427                  outer problem's; requires `pareto_front` to be set in the
 428                  problem description dictionaries, otherwise the value of this
 429                  indicator will be `NaN`; `rg_n_eval` should also be set in
 430                  the problem description, but defaults to 1 if not;
 431                * `rggd+`: resampled ground generational distance plus;
 432                  requires `pareto_front` to be set in the problem description
 433                  dictionaries, otherwise the value of this indicator will be
 434                  `NaN`; `rg_n_eval` should also be set in the problem
 435                  description, but defaults to 1 if not; `rg_n_eval` should
 436                  also be set in the problem description, but defaults to 1 if
 437                  not;
 438                * `rghv`: resampled ground hypervolume; requires `hv_ref_point`
 439                  to be set in the problem discription dictionaries, otherwise
 440                  the value of this indicator will be `NaN`; `rg_n_eval` should
 441                  also be set in the problem description, but defaults to 1 if
 442                  not;
 443                * `rgigd`: resampled ground inverted generational distance;
 444                  requires `pareto_front` to be set in the problem description
 445                  dictionaries, otherwise the value of this indicator will be
 446                  `NaN`; `rg_n_eval` should also be set in the problem
 447                  description, but defaults to 1 if not;
 448                * `rgigd+`: resampled ground inverted generational distance
 449                  plus; requires `pareto_front` to be set in the problem
 450                  description dictionaries, otherwise the value of this
 451                  indicator will be `NaN`; `rg_n_eval` should also be set in
 452                  the problem description, but defaults to 1 if not.
 453
 454                In the result dataframe, the corresponding columns will be
 455                named `perf_<name of indicator>`, e.g. `perf_igd`. If left
 456                unspecified, defaults to `["igd"]`.
 457
 458            seeds (Optional[List[Optional[int]]]): List of seeds to use. The
 459                first seed will be used for the first run of every
 460                algorithm-problem pair, etc.
 461        """
 462        self._output_dir_path = Path(output_dir_path)
 463        self._set_problems(problems)
 464        self._set_algorithms(algorithms)
 465        if n_runs <= 0:
 466            raise ValueError(
 467                "The number of run (for each problem-algorithm pair) must be "
 468                "at least 1."
 469            )
 470        self._n_runs = n_runs
 471        self._dump_histories = dump_histories
 472        self._set_performance_indicators(performance_indicators)
 473        self._max_retry = max_retry
 474        if seeds is None:
 475            self._seeds = [None] * n_runs
 476        elif len(seeds) < n_runs:
 477            raise ValueError(
 478                f"Not enough seeds: provided {len(seeds)} seeds but specified "
 479                f"{n_runs} runs."
 480            )
 481        else:
 482            if len(seeds) > n_runs:
 483                logging.warning(
 484                    "Too many seeds: provided {} but only need {} "
 485                    "(i.e. n_run)",
 486                    len(seeds),
 487                    n_runs,
 488                )
 489            self._seeds = seeds

 491    def _compute_global_pareto_population(self, pair: PAPair) -> None:
 492        """
 493        Computes the global Pareto population of a given problem-algorithm
 494        pair. See `compute_global_pareto_populations`. Assumes that the global
 495        Pareto population has not already been calculated, i.e. that
 496        `<output_dir_path>/<problem>.<algorithm>.gpp.npz` does not exist.
 497        """
 498        configure_logging(prefix=f"[{pair}]")
 499        logging.debug("Computing global Pareto population")
 500        gpp_path = (
 501            self._output_dir_path / pair.global_pareto_population_filename()
 502        )
 503        populations: Dict[str, List[np.ndarray]] = {}
 504        for n_run in range(1, self._n_runs + 1):
 505            triple = PARTriple(
 506                algorithm_description=pair.algorithm_description,
 507                algorithm_name=pair.algorithm_name,
 508                n_run=n_run,
 509                problem_description=pair.problem_description,
 510                problem_name=pair.problem_name,
 511            )
 512            path = self._output_dir_path / triple.pareto_population_filename()
 513            if not path.exists():
 514                logging.debug(
 515                    "File {} does not exist. The corresponding triple runs "
 516                    "most likely have not finished or all failed",
 517                    path,
 518                    triple,
 519                )
 520                continue
 521            data = np.load(path, allow_pickle=True)
 522            for k, v in data.items():
 523                populations[k] = populations.get(k, []) + [v]
 524            data.close()
 525
 526        consolidated = {k: np.concatenate(v) for k, v in populations.items()}
 527        if "F" not in consolidated:
 528            logging.error(
 529                "No Pareto population file found. The corresponding triple "
 530                "runs most likely have not finished or all failed",
 531            )
 532            return
 533        mask = pareto_frontier_mask(consolidated["F"])
 534        np.savez_compressed(
 535            gpp_path,
 536            **{k: v[mask] for k, v in consolidated.items()},
 537        )

 539    # pylint: disable=too-many-branches
 540    def _compute_performance_indicator(
 541        self, triple: PARTriple, pi_name: str
 542    ) -> None:
 543        """
 544        Computes a performance indicators for a given problem-algorithm-(run
 545        number) triple and stores it under
 546        `<problem_name>.<algorithm_name>.<n_run>.pi-<pi_name>.csv`. Assumes
 547        that the this performance indicator has not already been calculated,
 548        i.e. that that file does not exist.
 549
 550        Warning:
 551            This fails if either the top layer history or the pareto population
 552            artefact (`<problem_name>.<algorithm_name>.<n_run>.pp.npz`) could
 553            not be loaded as numpy arrays.
 554
 555        Todo:
 556            Refactor (again)
 557        """
 558        configure_logging(prefix=f"[{triple}|{pi_name}]")
 559        logging.debug("Computing PI")
 560
 561        pic: _PIC = lambda _: np.nan
 562        if pi_name == "df":
 563            problem = triple.problem_description["problem"]
 564            n_evals = triple.problem_description.get("df_n_evals", 1)
 565            delta_f = DeltaF(problem, n_evals)
 566            pic = lambda s: delta_f.do(s["F"], s["X"])
 567        elif pi_name in ["gd", "gd+", "igd", "igd+"]:
 568            pic = self._get_pic_gd_type(triple, pi_name)
 569        elif pi_name in ["ggd", "ggd+", "gigd", "gigd+"]:
 570            pic = self._get_pic_gd_type(triple, pi_name[1:])
 571        elif pi_name in ["rggd", "rggd+", "rgigd", "rgigd+"]:
 572            pic = self._get_pic_gd_type(triple, pi_name[2:])
 573        elif (
 574            pi_name in ["hv", "ghv", "rghv"]
 575            and "hv_ref_point" in triple.problem_description
 576        ):
 577            ref_point = triple.problem_description["hv_ref_point"]
 578            pi = get_performance_indicator("hv", ref_point=ref_point)
 579            pic = lambda s: pi.do(s["F"])
 580        elif pi_name == "ps":
 581            pic = lambda s: s["X"].shape[0]
 582        else:
 583            logging.warning(
 584                "Unprocessable performance indicator. This could be because "
 585                "required data is missing."
 586            )
 587
 588        # On which history is the PIC going to be called? By default, it is on
 589        # the top layer history.
 590        if pi_name in ["ps"]:
 591            history = np.load(
 592                self._output_dir_path / triple.pareto_population_filename()
 593            )
 594        elif pi_name in ["ggd", "ggd+", "ghv", "gigd", "gigd+"]:
 595            history = np.load(
 596                self._output_dir_path
 597                / triple.innermost_layer_history_filename()
 598            )
 599        elif pi_name in ["rggd", "rggd+", "rghv", "rgigd", "rgigd+"]:
 600            history = self._get_rg_history(triple)
 601        else:
 602            history = np.load(
 603                self._output_dir_path / triple.top_layer_history_filename()
 604            )
 605
 606        states: List[Dict[str, np.ndarray]] = []
 607        for i in range(1, history["_batch"].max() + 1):
 608            idx = history["_batch"] == i
 609            states.append({"X": history["X"][idx], "F": history["F"][idx]})
 610        df = pd.DataFrame()
 611        df["perf_" + pi_name] = list(map(pic, states))
 612        df["algorithm"] = triple.algorithm_name
 613        df["problem"] = triple.problem_name
 614        df["n_gen"] = range(1, len(states) + 1)
 615        df["n_run"] = triple.n_run
 616        logging.debug(
 617            "Writing result to {}",
 618            self._output_dir_path / triple.pi_filename(pi_name),
 619        )
 620        df.to_csv(self._output_dir_path / triple.pi_filename(pi_name))
 621
 622        if pi_name == "df":
 623            logging.debug(
 624                "Writing delta F history to {}",
 625                self._output_dir_path
 626                / triple.denoised_top_layer_history_filename(),
 627            )
 628            delta_f.dump_history(
 629                self._output_dir_path
 630                / triple.denoised_top_layer_history_filename()
 631            )

 633    def _get_pic_gd_type(self, triple: PARTriple, pi_name: str) -> _PIC:
 634        """
 635        Returns the `_PIC` corresponding to the either the `gd`, `gd+`, `igd`,
 636        or `igd+` performance indicator. As a reminder, a `_PIC`, or
 637        Performance Indicator Callable, is a function that takes a dict of
 638        `np.ndarray` and returns an optional `float`. In this case, the dict
 639        must have the key `F`.
 640        """
 641        if "pareto_front" in triple.problem_description:
 642            pf = triple.problem_description.get("pareto_front")
 643        else:
 644            path = (
 645                self._output_dir_path
 646                / triple.global_pareto_population_filename()
 647            )
 648            data = np.load(path)
 649            pf = data["F"]
 650            data.close()
 651        pi = get_performance_indicator(pi_name, pf)
 652        return lambda s: pi.do(s["F"])

 654    def _get_rg_history(self, triple: PARTriple) -> Dict[str, np.ndarray]:
 655        """
 656        Returns the `X` and `F` history of the ground problem of the triple,
 657        but where `F` has been resampled a given number of times (`rg_n_evals`
 658        parameter in the problem's description). This involves wrapping the
 659        ground problem in a `nmoo.denoisers.ResampleAverage` and evaluating the
 660        history's `X` array.
 661        """
 662        history = dict(
 663            np.load(
 664                self._output_dir_path / triple.top_layer_history_filename()
 665            )
 666        )
 667        rgp = ResampleAverage(
 668            triple.problem_description["problem"].ground_problem(),
 669            triple.problem_description.get("rg_n_eval", 1),
 670        )
 671        history["F"] = rgp.evaluate(history["X"], return_values_of="F")
 672        return history

 682    def _run_par_triple(
 683        self,
 684        triple: PARTriple,
 685    ) -> None:
 686        """
 687        Runs a given algorithm against a given problem. See
 688        `nmoo.benchmark.Benchmark.run`. Immediately dumps the history of the
 689        problem and all wrapped problems with the following naming scheme:
 690
 691            output_dir_path/<problem_name>.<algorithm_name>.<n_run>.<level>.npz
 692
 693        where `level` is the depth of the wrapped problem, starting at `1`. See
 694        `nmoo.wrapped_problem.WrappedProblem.dump_all_histories`. It also dumps
 695        the compounded Pareto population for every at every generation (or just
 696        the last generation of `set_history` is set to `False` in the algorithm
 697        description) in
 698
 699            output_dir_path/<problem_name>.<algorithm_name>.<n_run>.pp.npz
 700
 701        Additionally, it generates a CSV file containing various statistics
 702        named:
 703
 704            output_dir_path/<problem_name>.<algorithm_name>.<n_run>.csv
 705
 706        The existence of this file is also used to determine if the
 707        problem-algorithm-(run number) triple has already been run when
 708        resuming a benchmark.
 709
 710        Args:
 711            triple: A `PARTriple` object representing the
 712                problem-algorithm-(run number) triple to run.
 713        """
 714        configure_logging(prefix=f"[{str(triple)}]")
 715        logging.info("Starting run")
 716
 717        triple.problem_description["problem"].start_new_run()
 718        evaluator = triple.problem_description.get(
 719            "evaluator",
 720            triple.algorithm_description.get("evaluator"),
 721        )
 722        try:
 723            seed = self._seeds[triple.n_run - 1]
 724            problem = deepcopy(triple.problem_description["problem"])
 725            problem.reseed(seed)
 726            results = minimize(
 727                problem,
 728                triple.algorithm_description["algorithm"],
 729                termination=triple.algorithm_description.get("termination"),
 730                copy_algorithm=True,
 731                copy_termination=True,
 732                # extra Algorithm.setup kwargs
 733                callback=TimerCallback(),
 734                display=triple.algorithm_description.get("display"),
 735                evaluator=deepcopy(evaluator),
 736                return_least_infeasible=triple.algorithm_description.get(
 737                    "return_least_infeasible", False
 738                ),
 739                save_history=True,
 740                seed=seed,
 741                verbose=triple.algorithm_description.get("verbose", False),
 742            )
 743        except Exception as e:  # pylint: disable=broad-except
 744            logging.error("Run failed: {}", e)
 745            return
 746        else:
 747            logging.success("Run successful")
 748
 749        # Dump all layers histories
 750        logging.debug("Writing layer histories to {}", self._output_dir_path)
 751        if self._dump_histories:
 752            results.problem.dump_all_histories(
 753                self._output_dir_path,
 754                triple.filename_prefix(),
 755            )
 756
 757        # Dump pareto sets
 758        logging.debug(
 759            "Writing pareto sets to {}",
 760            self._output_dir_path / triple.pareto_population_filename(),
 761        )
 762        np.savez_compressed(
 763            self._output_dir_path / triple.pareto_population_filename(),
 764            **population_list_to_dict([h.opt for h in results.history]),
 765        )
 766
 767        # Create and dump CSV file
 768        logging.debug(
 769            "Writing result CSV to {}",
 770            self._output_dir_path / triple.result_filename(),
 771        )
 772        df = pd.DataFrame()
 773        df["n_gen"] = [a.n_gen for a in results.history]
 774        df["n_eval"] = [a.evaluator.n_eval for a in results.history]
 775        df["timedelta"] = results.algorithm.callback._deltas
 776        # Important to create these columns once the dataframe has its full
 777        # length
 778        df["algorithm"] = triple.algorithm_name
 779        df["problem"] = triple.problem_name
 780        df["n_run"] = triple.n_run
 781        df.to_csv(
 782            self._output_dir_path / triple.result_filename(),
 783            index=False,
 784        )

 911    def consolidate(self) -> None:
 912        """
 913        Merges all statistics dataframes
 914        (`<problem_name>.<algorithm_name>.<n_run>.csv`) and all PI dataframes
 915        (`<problem_name>.<algorithm_name>.<n_run>.pi.csv`) into a single
 916        dataframe, and saves it under `output_dir_path/benchmark.csv`.
 917        """
 918        logging.info("Consolidating statistics")
 919        all_df = []
 920        for triple in self.all_par_triples():
 921            path = self._output_dir_path / triple.result_filename()
 922            if not path.exists():
 923                logging.debug(
 924                    "Statistic file {} does not exist. The corresponding "
 925                    "triple [{}] most likely hasn't finished or failed",
 926                    path,
 927                    triple,
 928                )
 929                continue
 930            all_df.append(pd.read_csv(path))
 931        self._results = pd.concat(all_df, ignore_index=True)
 932        self._results["timedelta"] = pd.to_timedelta(
 933            self._results["timedelta"]
 934        )
 935        self._results = self._results.astype(
 936            {
 937                "algorithm": "category",
 938                "n_gen": "uint32",
 939                "n_run": "uint32",
 940                "problem": "category",
 941            }
 942        )
 943
 944        logging.info("Consolidating performance indicators")
 945        all_df = []
 946        for triple in self.all_par_triples():
 947            df = pd.DataFrame()
 948            for pi_name in self._performance_indicators:
 949                path = self._output_dir_path / triple.pi_filename(pi_name)
 950                if not path.exists():
 951                    logging.debug("PI file {} does not exist.", path)
 952                    continue
 953                tmp = pd.read_csv(path)
 954                if df.empty:
 955                    df = tmp
 956                else:
 957                    col = "perf_" + pi_name
 958                    df[col] = tmp[col]
 959            all_df.append(df)
 960
 961        self._results = self._results.merge(
 962            pd.concat(all_df, ignore_index=True),
 963            how="outer",
 964            on=["algorithm", "problem", "n_gen", "n_run"],
 965        )
 966
 967        # ???
 968        if "Unnamed: 0" in self._results:
 969            del self._results["Unnamed: 0"]
 970
 971        path = self._output_dir_path / "benchmark.csv"
 972        logging.info("Writing results to {}", path)
 973        self.dump_results(path, index=False)

 582            pic = lambda s: s["X"].shape[0]
 583        else:
 584            logging.warning(
 585                "Unprocessable performance indicator. This could be because "
 586                "required data is missing."
 587            )
 588
 589        # On which history is the PIC going to be called? By default, it is on
 590        # the top layer history.
 591        if pi_name in ["ps"]:
 592            history = np.load(
 593                self._output_dir_path / triple.pareto_population_filename()
 594            )
 595        elif pi_name in ["ggd", "ggd+", "ghv", "gigd", "gigd+"]:
 596            history = np.load(
 597                self._output_dir_path
 598                / triple.innermost_layer_history_filename()
 599            )
 600        elif pi_name in ["rggd", "rggd+", "rghv", "rgigd", "rgigd+"]:
 601            history = self._get_rg_history(triple)
 602        else:
 603            history = np.load(
 604                self._output_dir_path / triple.top_layer_history_filename()
 605            )
 606
 607        states: List[Dict[str, np.ndarray]] = []
 608        for i in range(1, history["_batch"].max() + 1):
 609            idx = history["_batch"] == i
 610            states.append({"X": history["X"][idx], "F": history["F"][idx]})
 611        df = pd.DataFrame()
 612        df["perf_" + pi_name] = list(map(pic, states))
 613        df["algorithm"] = triple.algorithm_name
 614        df["problem"] = triple.problem_name
 615        df["n_gen"] = range(1, len(states) + 1)
 616        df["n_run"] = triple.n_run
 617        logging.debug(
 618            "Writing result to {}",
 619            self._output_dir_path / triple.pi_filename(pi_name),
 620        )
 621        df.to_csv(self._output_dir_path / triple.pi_filename(pi_name))
 622
 623        if pi_name == "df":
 624            logging.debug(
 625                "Writing delta F history to {}",
 626                self._output_dir_path
 627                / triple.denoised_top_layer_history_filename(),
 628            )
 629            delta_f.dump_history(
 630                self._output_dir_path
 631                / triple.denoised_top_layer_history_filename()
 632            )

 655    def _get_rg_history(self, triple: PARTriple) -> Dict[str, np.ndarray]:
 656        """
 657        Returns the `X` and `F` history of the ground problem of the triple,
 658        but where `F` has been resampled a given number of times (`rg_n_evals`

 683    def _run_par_triple(
 684        self,
 685        triple: PARTriple,
 686    ) -> None:
 687        """
 688        Runs a given algorithm against a given problem. See
 689        `nmoo.benchmark.Benchmark.run`. Immediately dumps the history of the
 690        problem and all wrapped problems with the following naming scheme:
 691
 692            output_dir_path/<problem_name>.<algorithm_name>.<n_run>.<level>.npz
 693
 694        where `level` is the depth of the wrapped problem, starting at `1`. See
 695        `nmoo.wrapped_problem.WrappedProblem.dump_all_histories`. It also dumps
 696        the compounded Pareto population for every at every generation (or just
 697        the last generation of `set_history` is set to `False` in the algorithm
 698        description) in
 699
 700            output_dir_path/<problem_name>.<algorithm_name>.<n_run>.pp.npz
 701
 702        Additionally, it generates a CSV file containing various statistics
 703        named:
 704
 705            output_dir_path/<problem_name>.<algorithm_name>.<n_run>.csv
 706
 707        The existence of this file is also used to determine if the
 708        problem-algorithm-(run number) triple has already been run when
 709        resuming a benchmark.
 710
 711        Args:
 712            triple: A `PARTriple` object representing the
 713                problem-algorithm-(run number) triple to run.
 714        """
 715        configure_logging(prefix=f"[{str(triple)}]")
 716        logging.info("Starting run")
 717
 718        triple.problem_description["problem"].start_new_run()
 719        evaluator = triple.problem_description.get(
 720            "evaluator",
 721            triple.algorithm_description.get("evaluator"),
 722        )
 723        try:
 724            seed = self._seeds[triple.n_run - 1]
 725            problem = deepcopy(triple.problem_description["problem"])
 726            problem.reseed(seed)
 727            results = minimize(
 728                problem,
 729                triple.algorithm_description["algorithm"],
 730                termination=triple.algorithm_description.get("termination"),
 731                copy_algorithm=True,
 732                copy_termination=True,
 733                # extra Algorithm.setup kwargs
 734                callback=TimerCallback(),
 735                display=triple.algorithm_description.get("display"),
 736                evaluator=deepcopy(evaluator),
 737                return_least_infeasible=triple.algorithm_description.get(
 738                    "return_least_infeasible", False
 739                ),
 740                save_history=True,
 741                seed=seed,
 742                verbose=triple.algorithm_description.get("verbose", False),
 743            )
 744        except Exception as e:  # pylint: disable=broad-except
 745            logging.error("Run failed: {}", e)
 746            return
 747        else:
 748            logging.success("Run successful")
 749
 750        # Dump all layers histories
 751        logging.debug("Writing layer histories to {}", self._output_dir_path)
 752        if self._dump_histories:
 753            results.problem.dump_all_histories(
 754                self._output_dir_path,
 755                triple.filename_prefix(),
 756            )
 757
 758        # Dump pareto sets
 759        logging.debug(
 760            "Writing pareto sets to {}",
 761            self._output_dir_path / triple.pareto_population_filename(),
 762        )
 763        np.savez_compressed(
 764            self._output_dir_path / triple.pareto_population_filename(),
 765            **population_list_to_dict([h.opt for h in results.history]),
 766        )
 767
 768        # Create and dump CSV file
 769        logging.debug(
 770            "Writing result CSV to {}",
 771            self._output_dir_path / triple.result_filename(),
 772        )
 773        df = pd.DataFrame()
 774        df["n_gen"] = [a.n_gen for a in results.history]
 775        df["n_eval"] = [a.evaluator.n_eval for a in results.history]
 776        df["timedelta"] = results.algorithm.callback._deltas
 777        # Important to create these columns once the dataframe has its full
 778        # length
 779        df["algorithm"] = triple.algorithm_name
 780        df["problem"] = triple.problem_name
 781        df["n_run"] = triple.n_run
 782        df.to_csv(
 783            self._output_dir_path / triple.result_filename(),
 784            index=False,
 785        )

1046    def run(
1047        self,
1048        n_jobs: int = -1,
1049        n_post_processing_jobs: int = 2,
1050        **joblib_kwargs,
1051    ):
1052        """
1053        Runs the benchmark sequentially. Makes your laptop go brr. The
1054        histories of all problems are progressively dumped in the specified
1055        output directory as the benchmark run. At the end, the benchmark
1056        results are dumped in `output_dir_path/benchmark.csv`.
1057
1058        Args:
1059            n_jobs (int): Number of processes to use. See the `joblib.Parallel`_
1060                documentation. Defaults to `-1`, i.e. all CPUs are used.
1061            n_post_processing_jobs (int): Number of processes to use for post
1062                processing tasks (computing global Pareto populations and
1063                performance indicators). These are memory-intensive tasks.
1064                Defaults to `2`.
1065            joblib_kwargs (dict): Additional kwargs to pass on to the
1066                `joblib.Parallel`_ instance.
1067
1068        .. _joblib.Parallel:
1069            https://joblib.readthedocs.io/en/latest/generated/joblib.Parallel.html
1070        """
1071        if not os.path.isdir(self._output_dir_path):
1072            os.mkdir(self._output_dir_path)
1073        triples = self.all_par_triples()
1074        executor = Parallel(n_jobs=n_jobs, **joblib_kwargs)
1075        current_round = 0
1076        while (
1077            self._max_retry < 0 or current_round <= self._max_retry
1078        ) and any(not self._par_triple_done(t) for t in triples):
1079            executor(
1080                delayed(Benchmark._run_par_triple)(self, t)
1081                for t in triples
1082                if not self._par_triple_done(t)
1083            )
1084            current_round += 1
1085        if any(not self._par_triple_done(t) for t in triples):
1086            logging.warning(
1087                "Benchmark finished, but some triples could not be run "
1088                "successfully within the retry budget ({}):",
1089                self._max_retry,
1090            )
1091            for t in filter(lambda x: not self._par_triple_done(x), triples):
1092                logging.warning("    [{}]", t)
1093        self.compute_global_pareto_populations(
1094            n_post_processing_jobs, **joblib_kwargs
1095        )
1096        self.compute_performance_indicators(
1097            n_post_processing_jobs, **joblib_kwargs
1098        )
1099        self.consolidate()

Runs the benchmark sequentially. Makes your laptop go brr. The histories of all problems are progressively dumped in the specified output directory as the benchmark run. At the end, the benchmark results are dumped in output_dir_path/benchmark.csv.

Arguments:
  • n_jobs (int): Number of processes to use. See the joblib.Parallel documentation. Defaults to -1, i.e. all CPUs are used.
  • n_post_processing_jobs (int): Number of processes to use for post processing tasks (computing global Pareto populations and performance indicators). These are memory-intensive tasks. Defaults to 2.
  • joblib_kwargs (dict): Additional kwargs to pass on to the joblib.Parallel instance.
